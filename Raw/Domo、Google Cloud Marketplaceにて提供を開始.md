Domo、Google Cloud Marketplaceにて提供を開始ドーモ株式会社2026年4月28日 11時00分いいね！数を読み込み中です（米国報道発表資料抄訳）
Domo（本社：米国ユタ州シリコンスロープス）は、AIを搭載した全社データ活用プラットフォーム「Domo」がGoogle Cloud Marketplaceにて利用可能になったことを発表しました。これにより、企業はGoogle Cloud上でDomoをスムーズに導入・活用でき、AIをデータ基盤上で直接業務に組み込み、実運用できるようになります。本提供により、企業はクラウドや各種システム、オンプレミスなど、さまざまな環境に分散したデータを単一の統制されたプラットフォームで統合・整備・活用できるようになります。さらに、既存のGoogle Cloudの利用契約を活用することで、調達プロセスの簡素化も実現します。

DomoがGoogle Cloud Marketplaceに掲載されたことで、Google Cloud上ですぐにDomoを利用開始でき、既存の契約を活用してそのまま購入できます。これにより、導入の検討から実行までのプロセスが簡素化され、迅速な活用が進みます。また、企業はさまざまなデータソースをGoogle Cloudのデータ基盤BigQueryに連携し、全社のデータを一元化できます。さらに、クラウド上のデータを各部門で活用できる、ガバナンスの効いた、業務で使えるデータへと整備することが可能になります。

DomoはGoogle Cloud上で直接稼働し、BigQueryやVertex AI、Geminiと連携することで、データの収集から分析、AI活用に至るまでのプロセスを単一のプラットフォームで一元的に管理できる環境を提供します。1,000以上のコネクターを標準装備しているDomoは、クラウドやオンプレミスなど多様なデータソースからデータを取り込むことが可能です。さらに、ノーコードでデータ加工を行える「Magic ETL」により、Google Cloudに最適化されたSQLを自動生成しながらデータを整備できます。これにより、データを複製することなくBigQuery上のデータをそのまま活用し、実用的なデータ活用基盤や分析環境を迅速に構築できます。

多くの企業がAIを個別ツールで試行する中、Domoは統制されたGoogle BigQuery上のデータとAIをシームレスに連携し、高度な分析や意思決定機能を業務プロセスへ直接組み込むことを可能にします。さらに、人の関与を前提としたエージェント運用により、クラウド上のデータを「分析」で終わらせることなく、実際の業務に活かし、ビジネス成果へとつなげます。これにより、チームは業務の自動化を推進しながらリアルタイムにインサイトを把握し、ガバナンスの効いたエンタープライズ環境で安心してAIを活用できます。また、人が関与し続ける仕組みによって、継続的なコントロールと信頼性を確保した運用が実現します。

■Domoの創業者兼CEO、ジャシュ ジェイムズのコメント
DomoがGoogle Cloud Marketplaceで利用可能になったことで、クラウド上のデータをより迅速にビジネス成果へとつなげることができます。企業は既存のGoogle Cloudの利用契約を活用しながら、BigQuery上でDomoをすぐに使い始めることができ、複雑な手間をかけることなく、データをビジネスのアクションへと結び付けることが可能になります。
 
■‍Google CloudのマーケットプレイスおよびISV GTMプログラム担当マネージングディレクター、Dai Vu氏のコメント
Google Cloud MarketplaceでDomoを提供することで、Google Cloudの信頼性の高いグローバル基盤上で、お客様はAIを搭載した全社データ活用プラットフォームを迅速に導入・運用し、ビジネスの成長に合わせて拡張できるようになります。また、Domoを活用して、企業全体のデータを統合し、クラウド上のデータを意思決定や業務にすぐ活用できる形に整え、安全かつ柔軟に拡張することが可能になります。

ドーモ株式会社について
ドーモ株式会社は、AIを搭載した全社データ活用プラットフォーム「Domo」を企業向けに提供しています。Domoは、データの統合・可視化・自動化・共有化、そして埋め込みアプリケーション作成など、データ活用に必要な機能をワンストップで提供するプラットフォームです。企業における既存もしくは新規データ環境に柔軟に対応し、散在するデータを一元管理することで、企業のデータ価値を最大限に引き出し、ビジネス成果を最大化します。Domoに搭載しているAIサービスDomo.AIは、専門知識がなくても誰もがスピーディかつ的確に複雑なデータを読み解けるようにサポートし、すべての従業員のデータ活用を効率化して、迅速な意思決定を支援します。Domoは、データに基づくアクションを必要としている、あらゆる事業規模・業界で活用されています。

ドーモ株式会社では、DX成功に導くために、データ活用を全社に展開し企業文化として根付かせる重要なポジションとして、新たな役職「データアンバサダー」を提唱しています。ドーモはこれからも企業のDX推進に寄り添い、ナレッジベース、トレーニング、コミュニティ活動を通じて、有益な情報を発信し続けると共に、企業のDX化を推進する「データアンバサダー」の活動を全面的に支援していきます。
※ドーモ株式会社が提唱するDXを成功に導くための必須人材「データアンバサダー」について
ドーモ株式会社が提唱するデータアンバサダーは、DXの推進、データドリブンなビジネス環境の実現、データ活用に向けた企業文化の醸成に責任を持ち、データ人材と言われるデータエンジニア、データサイエンティスト、データアナリストではカバーできなかった新たな職務を担います。
データアンバサダーは、「経営者」「事業部門」「IT部門」のそれぞれが抱えるデータ活用の課題を理解し、三者の橋渡し役として企業全体のデータ活用の能力を高めていくミッションがあります。
日本企業においても、早い段階で「データアンバサダー」を任命し、変化の激しい市場環境に勝ち残れる体制を構築することで、より早くDXを軌道に乗せることができます。
