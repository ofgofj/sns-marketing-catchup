クラウドストライク（NASDAQ：CRWD）は本日、新しいデータセキュリティソリューション、CrowdStrike Falcon Data Securityを発表しました。これにより、エージェント型AIを導入した企業全体にわたり、データ窃取を阻止します。実際の攻撃者インテリジェンスとFalcon®プラットフォームの統合コンテキストを活用したFalcon Data Securityは、従業員のミスや悪意のあるインサイダー、正規の認証情報を悪用した攻撃者など、いずれに起因するものであるかを問わず、データ窃取をリアルタイムで検出、分類、阻止します。

クラウドストライクの最高技術責任者を務めるエリア・ザイツェフ（Elia Zaitsev）は次のように述べています。
「AIはデータの生成、アクセス、共有のスピードを根本的に変えてしまったため、機密情報は常に移動している状態にあります。従来のデータ保護ツールは、データが静的だった環境を前提に構築されていました。Falcon Data Securityは、機密データがどこに存在しようと、静止状態でも移動中でも、データを検出、分類、保護します。これこそがエージェント型AI時代のデータセキュリティです」

移動するデータAIは業務の遂行方法を一変させ、エンドポイント、クラウドサービス、SaaSアプリケーション、AIを活用したワークフローをまたいで絶えず機密データにアクセスし、移動させています。従来のデータ保護は、現在とは異なる時代を想定して構築されており、静的なエンドポイント向けのDLPや、特定時点のクラウドインベントリーを対象としたDSPMで対応していました。DLPもDSPMも、機密情報がドメイン間を移動するような包括的なデータセキュリティを想定して設計されていません。最新のデータセキュリティでは、正規のデータ移動と実際のリスクを区別し、データがどこに存在していようと意図しない情報の露出、データ漏洩、盗難を防ぐために、リアルタイムのクロスドメインコンテキストと攻撃者インテリジェンスが必要です。

統合型のデータセキュリティFalconプラットフォームの統合アーキテクチャは、アイデンティティ、エンドポイント、SaaS、クラウド、AIまでを網羅し、ネイティブの攻撃者インテリジェンスと組み合わせて、システム全体で機密データを検出、分類、保護するためのコンテキストをチームに提供します。断片化されたDLPツールとDSPMツールを統合することで、Falcon Data Securityは保護を簡素化し、データ領域全体を保護します。

主要な新機能と利点は次のとおりです。


AIを活用した移動中のデータの分類：エンドポイント、SaaS、クラウド、ブラウザ、AIワークフローまでを対象に、機密データの作成、変換、共有時にデータを自動的に識別して分類します。 


生成AIデータ保護：ブラウザ、ローカルアプリケーション、ランタイムクラウド環境を通じて共有される機密データを保護します。管理対象および管理対象外の生成AIツールがどこで使用されても、意図しない情報の露出を防ぎながらデータ漏洩をブロックします。


ランタイムクラウドデータの可視性：静的なインベントリーを超えて、機密データがどのようにアクセスされ、移動しているかについてリアルタイムのインサイトを提供するため、アクティブなクラウドデータのリスクを明らかにします。


クロスドメインプラットフォームのコンテキスト：エンドポイント、アイデンティティ、およびクラウドアクティビティに沿ってデータの脅威を評価します。これにより、チームはユーザー、デバイス、アクセスに関するすべてのコンテキストを単一のコンソールで取得できるため、より迅速に脅威を相関付け、優先順位を付けて阻止できます。


自動化された強制：エンドポイントでの危険な外部へのデータ移動をブロックし、SaaSアプリケーションでの不正アクセスを防止し、統合SOARワークフローを通じてクラウドデータ対応アクションを自動化します。
