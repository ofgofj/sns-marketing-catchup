ノーコードでAIアプリ開発を加速する「Maira」提供のGigalogy、「Web Summit Vancouver 2026」に出展Gigalogy株式会社2026年4月28日 10時00分いいね！数を読み込み中ですGigalogy株式会社（本社：東京都台東区、代表取締役 ウッディン・エムディー・モスレ、以下ギガロジ）は、2026年5月11日(月)から14日(木)にかけてカナダ・バンクーバーで開催される世界的テクノロジーカンファレンス Web Summit Vancouver 2026 に出展いたします。

ギガロジは、これまでポルトガル・リスボンで開催された Web Summit に3度出展しており、バンクーバー開催への参加は今回が初めてとなります。今回はGrowth企業として参加し、当社が開発・提供する統合AIアプリケーションプラットフォーム「Maira」をご紹介いたします。

米国・サンフランシスコを拠点に北米市場への展開を進めるギガロジにとって、今回の出展は現地企業やパートナーとの新たな接点を創出し、事業拡大を加速させる重要な機会と位置づけています。
Web Summit 公式サイトより
Maira
「Maira（マイラ）」は、先進的な生成AIを活用し、ノーコードでスマートアプリケーションを数時間以内に構築できる統合AIプラットフォームです。直感的なUI設計により、専門的なプログラミング知識がなくても簡単に操作でき、企業ごとの業務課題に応じた高度なアプリケーションをスピーディに開発できます。

本展示では、Mairaの最新機能および新サービスをいち早くご紹介いたします。
1. 新機能：「maira-2」により、さらに高性能・高効率へ
時系列処理や統計解析に強みを持つモデルとして2025年10月にリリースされた「maira-1」は、このたび、より高速かつ高精度な応答を実現する次世代モデル「maira-2」へと進化しました。
「maira-2」では、最適化されたトークン管理により、リソースの効率的な活用とコスト管理を両立。さらに、利用シーンや質問内容に応じて最適なLLMを自動選択することで、より的確で質の高い回答を提供します。
2. 新サービス：個人向けAIサービスを提供開始
誰でも気軽に、自分専用のAIアシスタント開発・構築ができる個人向けAIプラットフォームの提供を開始します。
説明書を読む手間や複雑な初期設定を必要としないシンプルなUIにより、直感的な操作ですぐに利用可能です。さらに、「maira-2」搭載により、常に最適な応答体験を提供します。
導入は簡単3ステップ。利用目的に応じて、3つのプランをご用意しています。


Liteプラン
シンプルにAIとの会話を楽しめる、基本機能に特化したプランです。
あらかじめAIにどんな答え方をしてほしいかを設定し、意図に沿った形で、一般知識を使った質問対応や文章作成、アイデア出しができます。


Standardプラン
Liteプランに加え、資料を見せながらAIと会話できるプランです。 
質問と一緒にファイルを読み込ませると、その内容を踏まえて質問に答えたり、要点を整理したりできます。


Proプラン
Standardプランに加え、会話とデータを蓄積しながら使えるプランです。
過去にアップロードしたファイルや会話履歴をAIが活用することで背景を理解した応答が得られます。
