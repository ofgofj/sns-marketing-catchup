都市は資源になる——繊維廃棄物を「都市森林」として再定義PANECO®が「都市森林 × 都市循環」を統合し、資源循環インフラの社会実装を始動繊維資源の供給と循環を接続し、廃棄を前提としない資源構造の構築を開始株式会社ワークスタジオ2026年4月27日 19時34分いいね！数を読み込み中です繊維資源循環プラットフォーム「PANECO®」を展開する株式会社ワークスタジオ（代表取締役 原和広）は、都市に存在する繊維廃棄物を資源供給源として再定義する「都市森林」と、資源を循環させる「都市循環」を統合し、資源循環インフラの社会実装を始動しました。

本取り組みでは、企業・施設等から発生する繊維資源を対象に、回収・再資源化・製品供給を軸とし、将来的な再回収・再資源化へ接続可能な循環構造を構築。

外部資源に依存しない供給と循環の基盤形成を進め、資源安全保障を支える新たな社会インフラとしての展開を加速します。 

背景｜資源調達構造の限界と転換
世界的に資源価格の変動や供給不安が常態化する中、資源の安定確保は、社会および企業活動の持続性を左右する重要課題となっています。 

従来の資源調達構造は、 
・原料価格の変動   
・特定地域への供給依存   
・物流コストの上昇および供給遅延   
といった不確実性を内包しています。 

また、   
「調達 → 使用 → 廃棄」   
というリニア型の経済モデルは、すでに構造的な限界に達しています。
 
こうした状況の中、外部から資源を調達する構造から、自ら資源を生み出し、循環させる構造への転換が求められています。

今回の始動｜「都市森林 × 都市循環」の統合
PANECO®は、 
・都市に存在する繊維廃棄物を資源供給源として再定義する「都市森林」   
・資源を循環させる仕組みである「都市循環」   
を統合し、資源循環インフラの社会実装を始動しました。 

本取り組みでは、企業・施設等から発生する繊維資源を対象に、 
回収 → 再資源化 → 製品供給   
までのプロセスを軸とし、将来的な再回収・再資源化を含めた循環への接続を前提とした構造を構築しています。 

なお、繊維資源の回収については当社単独での主体運営ではなく、企業・回収事業者等との連携により実施しています。 

供給と循環は分断された機能ではなく、接続された資源構造として機能し始めています。

構造｜循環を前提とした資源モデル

この統合により、資源の捉え方は大きく変化しています。

 従来：   調達 → 使用 → 廃棄  
 PANECO®：   回収 → 再資源化 → 供給 → 使用   

さらに、将来的には   
再回収 → 再資源化   
へと接続していくことを前提としています。 

資源は消費されるものではなく、循環し続けることを前提とした設計へと転換しています。





実装｜「PANECO® board S」による社会実装
PANECO®では、繊維廃棄物を再資源化した建材「PANECO® board S」を通じて、すでに社会実装を進めてきました。 

本ボードは、 
・建材   
・内装材（床材を含む）   
・家具   
・プロダクト   
として、商業施設・オフィス・公共空間において活用されています。 

今回の取り組みでは、これらの製品についても、将来的な再資源化を前提とした設計思想のもと、循環構造への接続を進めています。

PANECO® board S

運用｜分断しない資源構造へ
PANECO®の特徴は、 
回収 → 再資源化 → 製品化 → 使用   
というプロセスを分断せず捉え、さらにその先の再資源化へと接続可能な構造を設計している点にあります。 

これは従来の一方向のリサイクルとは異なり、循環を前提とした資源設計です。

供給能力｜量産を前提とした構造
PANECO®は、資源循環を概念にとどめることなく、量産を前提とした供給構造の構築を進めています。
 
再資源化された繊維は「PANECO® board M」として供給され、建材・内装材・家具・プロダクトとして継続的に社会へ展開されます。 

さらに、ボードの量産は単なる生産拡大ではなく、資源循環をスケールさせるための社会実装そのものとして位置づけられています。 
資源循環は小規模な取り組みでは成立せず、供給可能な規模へと拡張されることで、初めて社会の中で機能します。 

PANECO®は、量産によるスケールアップを通じて、資源循環を社会に定着させる構造の実装を目指しています。 

また、PANECO®の思想および技術は、開発段階から一貫して、スケールアップおよび社会実装を前提として設計されています。 
個別の技術や製品にとどまらず、社会の中で機能する資源循環構造として成立することを前提に構築されています。

廃棄の再定義
本構造では、従来廃棄されていた繊維や製品についても、可能な限り資源として再活用することを前提としています。 

廃棄は終点ではなく、 次の資源化プロセスへと接続されるものへと再定義されます。


都市｜資源循環インフラへの転換
この取り組みにより、都市は 消費の場から、資源を生み出し循環させる基盤へと転換していきます。

都市は「資源を消費する場所」ではなく、資源を生成し循環させるインフラとして機能していきます。



社会的意義｜資源安全保障
本取り組みは、外部資源に依存しない供給構造の構築を目指すものであり、資源安全保障の観点からも重要な意味を持ちます。 

資源は外部から調達するものではなく、都市の中で生み出し、循環させていくものへと変化しています。

スケール｜社会インフラへの拡張
PANECO®は、この資源循環インフラについて、都市単位での実装を見据え、社会全体への展開を進めていきます。 

資源循環は個別の取り組みではなく、社会の前提となるインフラへと移行していきます。
